Changes to be made in the 'User Defined Variables' (each time the test cases are executed):
----------------------------------------------------------------------------------------
	Before running the test cases in 'P2P.jmx' these changes needs to be made in the user defined variable. All these changes should be made before each execution otherwise the test cases would fail.
	
	
MSISDN for Registration:
------------------------
	Here MSISDN for registring the user for P2P is defined under 'MSISDN_USSD_P2P'. This field needs to be changed with a valid MSISDN. 
	
Buddy Name:
-----------
	A name for the buddy needs to be updated, which can be found under 'BUDDY_NAME'. 

Buddy MSISDN:
------------
	The MSISDN of the Buddy needs to be updated under 'Buddy MSISDN' with a valid MSISDN.
